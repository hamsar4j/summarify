## Summarify

Summarify is a web extension that can be used to summarise webpages quickly using local LLMs, created with Vite and TailwindCSS.

### ollama setup

```console
launchctl setenv OLLAMA_ORIGINS "*"
ollama pull {model_name}
ollama serve
```
